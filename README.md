# 🐚 ShellScript

Repositório com scripts Bash para Linux — automação de tarefas, gerenciamento do sistema e utilitários gerais.

## 🚀 Como usar

Clone o repositório:
```bash
git clone https://github.com/seu-usuario/ShellScript.git
cd ShellScript
```

Dê permissão de execução e rode:
```bash
chmod +x script.sh
./script.sh
```

> Alguns scripts exigem `sudo` por manipularem usuários ou serviços do sistema.

## 🛠️ Requisitos

- Linux (Ubuntu, Debian ou similar)
- Bash 4.0+
